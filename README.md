# IoT-Control-relays
Dự án nhỏ giúp điểu khiển motor bơm nước và đèn cổng trước sân nhà 

Sử dụng NodeMCU và Blynk 

Library sử dụng là ESP8266WiFi và Blynk 
# Sơ đồ kết nối
- NodeMCU     ------- Modules
- D2          ---------------> Relay1
- D3          ---------------> Relay2
- D5          ---------------> Pushbutton1
- D6          ---------------> Pushbutton2
- D7          ---------------> Led1
- D8          ---------------> Led2
